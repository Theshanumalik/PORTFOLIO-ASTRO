# Rate My School

Rate My School is a web application that allows students to rate their schools and share their experiences with others. The application is built using the MERN stack (MongoDB, Express, React, Node.js) and is hosted on Heroku.

## Features
- User authentication
- User profiles
- School ratings and reviews
- Search functionality
- Pagination
- Image uploads
- Responsive design

## Technologies Used
- Node.js
- Express
- MongoDB
- React
- Firebase
- Heroku
